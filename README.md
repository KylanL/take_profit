# Take Profit ATR Dashboard

A lightweight Node.js dashboard for ATR-based take-profit and stop-loss tracking.

## Run Locally

```bash
npm start
```

Open:

```text
http://127.0.0.1:8769/index.html
```

## Deploy On Deno Deploy

Use this option if Render asks for a credit card.

1. Push this repository to GitHub.
2. Open `https://dash.deno.com`.
3. Create a new project from GitHub.
4. Select this repository.
5. Entrypoint: `main.ts`.
6. Deploy.

Open:

```text
https://your-project.deno.dev/index.html
```
